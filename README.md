1. Docker

### 1. Install Dependencies

```
yarn install
```

### 2. Make Docker Run

### 3. Run the Project

```
yarn start
```

### Others

#### Code Style Testing with ESLint

Coderbunker Wechaty Bot supports Airbnb's JavaScript Style by ESLint. To test if your code is with the right code style, you have to run ESLint: 

```
yarn run lint
```
